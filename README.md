# Raytracing in Rust
![header](assets/header.png)  

An implementation of Peter Shirley's Ray Tracing in One Weekend. This project was created with the sole purpose of improving my knowledge of the Rust language.

## How to build
```rust
cargo build
```

## Running the code
```rust
cargo run
```
